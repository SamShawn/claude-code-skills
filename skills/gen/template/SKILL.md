# Generate Template Skill

Generate code templates for common design patterns, algorithms, and data structures.

## When to Use

Use this skill when user needs:
- Design pattern implementations
- Algorithm templates
- Data structure implementations
- Common code patterns
- Learning examples

## Supported Patterns

### Creational Patterns
- Singleton (thread-safe)
- Factory Method
- Abstract Factory
- Builder
- Prototype

### Structural Patterns
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

### Behavioral Patterns
- Observer
- Strategy
- Command
- Iterator
- Mediator
- Memento
- State
- Template Method
- Visitor

### Data Structures
- Linked List
- Binary Tree
- Hash Table
- Stack/Queue
- Heap
- Trie
- LRU Cache

### Common Algorithms
- Sorting (QuickSort, MergeSort, etc.)
- Search (Binary Search, BFS, DFS)
- Dynamic Programming
- Greedy Algorithms
- Graph Algorithms

## Output Format

Provide:
1. Complete implementation
2. Usage example
3. Time/space complexity
4. When to use / avoid

## Language Support

Templates available in Python, JavaScript, TypeScript, Go, Java, and other languages.
